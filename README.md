# gt-map-edit

an open source external map editor for Geotastic with a lot of additional features.
**UNDER DEVELOPMENT**
## usage (うさげ)
### how 2 install
you need  python3
i won't tell you how to install it, ask chatgpt or something
```
python3 -m pip install -r requirements.txt
```
do --break-system-packages or set up an virtualenv if pip is being annoying
### how 2 self compile (not required)
```
make core minify
```
needs emscripten, terser

### how 2 run
```
make run
```
to run locally

if you're using a niche operating system for videogames (michealsoft binbows):
```
python3 app/server.py
```
